IPRED_A_docomo_2:
Frame rate: 30 fps
Picture size: 832x480
Spec version: HM10.1

(Category: intra coding; Sub-category: intra prediction)

The purpose of the stream is to exercise all intra prediction modes (35 modes for each of Y32x32, Y16x16, Y8x8, Y4x4, C16x16, C8x8, and C4x4 for a total of 245 modes).
The bitstream contains only I slices and one slice per picture. Checksum SEI messages are included.
