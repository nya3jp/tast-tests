Category: other coding tools; Sub-category: Persistent Rice Parameter

The purpose of the stream is to exercise the incrementing and decrementing of all 4 of the persistent Golomb Rice parameters, and the enabling/disabling of the tool.

The bitstream only contains I slices, and one slice per picture. It has been coded as GBR using the RExt profile: main-444-12-intra.

Checksum SEI messages are included.

Maximum transform skip log2 maximum size set to 5 (32x32) (previous revisions had the value of 8).

Level 3, main tier
Spec. version : HM16.3
